# Système de Caisse Restauration

## Présentation

Système de caisse moderne dédié à la restauration traditionnelle, rapide, à emporter et à la livraison.

Le projet est développé sous Android en Java et utilise Firebase pour la synchronisation temps réel des données entre les différents appareils d'un établissement.

L'objectif est de proposer une solution complète couvrant l'ensemble du cycle de vie d'une commande :

* Prise de commande
* Gestion des tables
* Gestion des produits
* Gestion des catégories
* Gestion des options
* Gestion des menus
* Écrans de contrôle cuisine et bar
* Impression réseau ESC/POS
* Gestion des utilisateurs et permissions
* Livraison
* Z Reporting
* Statistiques et analyses
* Configuration multi-appareils
